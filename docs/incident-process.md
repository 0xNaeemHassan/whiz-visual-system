# Incident Process

## Incident Lifecycle
1. Detect and triage.
2. Assign incident roles (IC, release owner, data owner, communications lead).
3. Contain impact and restore service.
4. Communicate status updates on a fixed cadence.
5. Resolve incident and complete post-incident review.
